Advanced Web Programming project implementing all Tutorial requirements (HTML/CSS + JavaScript + PHP/Database) # Student-Dashboard-
## Features

-  **Student Management** - Add, edit, delete students with validation
-  **Attendance Tracking** - 6 sessions with Present/Participated checkboxes
-  **Session Management** - Create & close attendance sessions
-  **Reports & Analytics** - Charts and statistics
-  **Form Validation** - Real-time client-side validation
-  **Database Integration** - MySQL with PDO
 Project Structure
```
student-dashboard/
├── index.php              # Homepage with statistics
├── students.php           # Student CRUD operations
├── attendance.php         # Take attendance for a session
├── sessions.php           # Session management
├── reports.php            # Statistics and charts
├── logout.php             # Logout page
├── save_attendance.php    # AJAX endpoint for saving
├── config.php             # Database configuration
├── db_connect.php         # Database connection
├── setup_database.sql     # Database schema
├── style.css              # Style 
└── script.js              # JavaScript utilities

Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript, jQuery
- **Backend**: PHP 8.x
- **Database**: MySQL
- **Charts**: Chart.js
- **Server**: WAMP/Apache
```
